# Huddle Landing Page

A responsive **Huddle landing page with a single introductory section**, built as a Frontend Mentor challenge using **HTML5 and CSS3**.

## 🚀 Overview
## 🔗[Project's Live Link]("https://landing-page-with-single-introducto.vercel.app/")
This project is a clean and responsive landing page design that adapts across **desktop, tablet, and mobile devices**.

It includes:

* A header with the Huddle logo
* A hero section with illustration and text
* A call-to-action **Register** button
* Social media links in the footer
* Responsive layouts using **media queries**

---

## 🛠️ Built With

* **HTML5** semantic structure
* **CSS3** custom properties
* **Flexbox** for layout
* **Media Queries** for responsiveness
* **Google Fonts**
* **SVG icons**

---

## 📱 Responsive Design

The project is optimized for:

* **Desktop** → large illustration + text side by side
* **Tablet** → stacked layout with better spacing
* **Mobile** → single column layout with centered text and CTA

Breakpoints used:

* `max-width: 699px` → Mobile
* `700px - 1200px` → Tablet
* `1200px+` → Desktop
